网站插件 Plugins

  1. Hexo插件 Live Reload (官方插件)

    Hexo允许在发布之前在本地浏览器进行调试预览，但是如果每次修改都需要反复执行“$ hexo g && hexo s”估计也很崩溃。Live Reload给力的地方在于它会自动完成上述步骤，要预览的时候直接F5刷新浏览器即可看到修改后的网页效果。当然，如果有时候修改后无法刷新出来的话，那就“$ hexo clean && hexo g && hexo s”重新来过。
